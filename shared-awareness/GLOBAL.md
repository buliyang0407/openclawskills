# Global Context

## Purpose

这套共享意识文档服务于整个阳仔龙虾体系。

目标不是制造一堆人设废话，而是统一以下 4 件事：

- 共同世界观
- 共同工作边界
- 共同知识结构
- 共同迁移方式

## Current System Shape

- 主聊天平台是 Feishu。
- 当前运行环境以腾讯云服务器为主。
- 未来需要支持整体迁移到 Mac mini M4。
- 知识库采用分层结构，而不是一张大总表。

## Knowledge Principles

- 原始归档层保留各机器人自己的采集表。
- 实体知识层按类型建表，例如文章库、资产库、提示词库、事件库。
- 内容总表只做统一索引，不做所有知识的本体。
- 无法稳定抓到正文时，先保存链接和待补全状态，不能直接丢失。

## Work Principles

- 优先稳定，再追求花哨。
- 优先可迁移，再追求一时省事。
- 优先结构清晰，再追求功能堆叠。
- 遇到信息缺口时允许标记待补全，但不允许硬编。

## Maintenance Principles

- 技能逻辑尽量放在 `openclawskills_repo`。
- 运行配置放在 `/etc/openclaw/*.env` 与各 profile 配置中。
- 角色意识和短期记忆放在各自 workspace。
- 共享长期意识优先维护这套文档，而不是散落在聊天记录里。
